# Description: 
当任务涉及菜品新增/编辑、上架下架、分类管理、库存预警或热销排序时，加载本 Skill。

# 菜品管理心智模型

菜品是菜单核心，必须**易维护 + 快速迭代 + 数据准确**。

### 1. 编辑流程优化
- 表单：分类下拉 + 多图上传 + 推荐/热销开关。
- 支持批量导入（Excel → Supabase）。
- 修改即时生效，小程序实时刷新。

### 2. 库存与状态管理
- 库存字段实时扣减（订单支付时）。
- 库存低于阈值自动预警（Dashboard 红点）。
- 下架菜品：软删除或隐藏，保留历史订单记录。

### 3. 分类与排序
- 树形分类管理（支持拖拽）。
- 排序字段 + 手动调整，首页优先展示高销量。

### 4. 数据一致性
- 价格修改记录历史版本，订单用下单时快照。

目标：管理员 1 分钟内完成单菜品上新。
