# 搜索与推荐心智模型

搜索不是“模糊匹配”，而是**提升下单率 + 个性化**。

### 1. 搜索功能哲学
- 全局搜索支持菜名/分类/标签，实时高亮匹配。
- 无结果时智能推荐相似或热销菜。
- 支持语音搜索（微信原生）。

### 2. 推荐策略
- 首页：热销 Top + 猜你喜欢（基于历史订单简单规则）。
- 菜品详情：相关推荐（同分类高销）。
- 未来可接 Supabase Edge Function 做简单协同过滤。

### 3. 性能优化
- 搜索用 Supabase full-text search 或 RPC。
- 推荐数据预计算缓存（每日更新）。

### 4. 运营控制
- 后台可手动置顶推荐位。
- A/B 测试支持（不同用户看到不同推荐）。

目标：搜索命中率 > 80%，推荐点击转化 > 20%。
